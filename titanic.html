The Titanic Project; Survival At Sea.

A novice data scientist, a green horn, a clueless beginner usually needs data to learn the skill and ultimately practice their newly acquired skills and the titanic data is one of a kind. The Titanic dataset is a classic beginner-friendly project in data science field. One of the first datasets many data science learners explore is the Titanic dataset. It's simple yet rich, and behind it lies a tragic, real-life story. The titanic dataset is one good in enriching the beginner with skills like understanding data, data cleaning (Exploratory Data Analysis), feature scaling, model building and evaluation..

Project Overview

The Titanic is a classic example of supervised machine learning task. Thus, in this analysis, my goal is to build a predictive model that answers the question "what people were more distinctly probable to survive than others?"

Libraries Used

  • Python
  • Pandas for data manipulation.
  • Scikit-Learn for building models
  • Matplotlib/Seaborn for visualization
  • Jupyter Notebok for analysis

Key Features

  • Age
  • Gender
  • Passenger class
  • Number of siblings
  • Fare
  • Cabin
  • Embarked

Results

The model achieved an accuracy of 76% using the logistics regression model. Visualizations also revealed suvival by passenger class and gender.
