Netflix Recommender System

A collaborative filtering system built to suggest movies users might enjoy based on similar user preferences.

Project Overview

In this project, I built a movie recommendation system using collaborative filtering techniques. The system suggests movies based on the preferences of users with similar viewing histories. This project established my EDA skils to a good level since it is a beginner friendly data analysis.

Libraries Used

  • Python (Pandas, NumPy)
  • Scikit-Learn
  • Surprise Library
  • Matplotlib/Seaborn

Key Features

  • Title (movie title)
  • Date (Includes day, month and year)

Results

The model achieved 85% top-10 accuracy for user recommendations. Visualizations also revealed strong genre-based clustering in user preferences.
