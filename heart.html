Heart Disease Prediction Using Machine Learning

Binary Heartbeat, where 1s and 0s Save Lives. Before the attack strikes, AI-Powered heart risk detection

Project Overview

This project explores the alliance between medicine and artificial intelligence, harnessing data-driven insights to refine diagnostics. The primary goal of the project is to develop a reliable model that can accurately predict the likelihood of individuals developing heart disease.

Libraries Used

  • Python (Pandas, NumPy)
  • Scikit-Learn
  • Numpy
  • Matplotlib/Seaborn
  • Scikit-Learn

Key Features

  • Age
  • Sex
  • Cholesterol
  • ChestPainType
  • RestingBP (RBP)
  • FastingBS (fbs)
  • Resting Electrocardiographic Results (restecg)
  • Maximum Heart Rate Achieved (MaxHR)
  • Exercise-Induced Angina (exang)
  • ST Depression Induced by Exercise (oldpeak)
  • Slope of Peak Exercise ST Segment (slope)
  • Target Variable (0 = No Heart Disease, 1 = heart disease)

Results

The logistic regression model achieved 84% accuracy while the K-Nearest Neighbour algorithm achieved 87% accuracy.
